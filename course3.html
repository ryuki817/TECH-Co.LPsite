ADOBEコース

コース概要

現代のWebアプリケーション開発に必要な技術を包括的に学習できるコースです。HTML/CSSの基礎から始まり、JavaScript、WordPress、Gitまで、Webデザイナーとして活躍するためのスキルを習得できます。

実際の開発現場で使用される最新の技術スタックを使用し、GitHub連携まで含めた実践的な開発フローを学習します。卒業時には即戦力として活躍できるレベルまでスキルを向上させます。

学習目標

  • HTML/CSSフレーム設計
  • JavaScriptフレームワークの習得
  • データベース設計と操作
  • バージョン管理とチーム開発
  • レスポンシブWebデザインの実装
HTML/CSS JavaScript WordPress Git GitHub
